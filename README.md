School Management System (SaaS)

This project is a School Management System designed as a multi-tenant platform, meaning each school’s data is kept separate and secure. It is built using React.js for the frontend and Node.js for the backend.

What It Does

The system helps schools manage all their core operations, including:

School Registration & Management: Add new schools and manage their information.

Class Management: Organize classes and assign teachers.

Student Management: Keep track of student records, including personal information and enrollment.

Teacher Management: Manage teachers’ profiles and assignments.

Attendance Tracking: Record and monitor attendance for students and teachers.

Payment Management: Track student payments and generate reports.

Exam Results & Grading: Enter, view, and analyze student exam results.

User Roles

The platform supports different types of users with different levels of access:

Super Admin: Can view and manage all schools, oversee the platform, and analyze overall data.

School Admin: Full control of their school, including classes, students, teachers, payments, and results.

Teacher: Can mark attendance and enter exam results.

Student: Can view personal attendance records, payments, and exam results.

Technical Overview

Backend: Node.js with Express, MongoDB for the database, JWT-based authentication, and input validation.

Frontend: React.js with Tailwind CSS, React Router for navigation, Axios for API requests, and Recharts for analytics.

Purpose

This system is intended to make school administration more organized, efficient, and easy to manage, all in a secure and role-based environment. Each school’s data is isolated to ensure privacy, and the system provides useful dashboards and reports for administrators, teachers, and students.
